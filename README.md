<h3 align="center">MachZero-R1X</h3>

- a powerful budget RC car made with features like Obstacle Avoidance, Bluetooth Connectivity, display animations & more made using Arduino uno, IR Infrared sensors, BO motors and a 1.8 inch screen.

----

## Features

* Obstacle Avoidance
* Complete 4 wheel control
* Bluetooth Connectivity
* Screen to display messages or animations.
